# TerraSustain
TerraSustain is a sustainability platform that addresses the issues of pollution by serving as a hub for connecting sustainability activists

## What it does
Terrasustain is an enhanced platform that empowers environmental and sustainability activists like NGOs, government authorities among others to get real time access to the information about pollution activities being carried out in the environment. It does by providing the reporting feature; where activists are able to upload different media (photos and videos) on recognized activities that are damaging the environment. 

### Key Features 
- Real time Satellite image analysis for deforestation identification 
- User Report about logging activities with proof (such as images or videos)
- Community engagement activities (Through demonstration of real time environment protection activities and sharing of tips for sustaining the environment)
- User engagement (Through gamification rankings because of certain activities)

### Key Structure
 TerraSnap is available in two main states
- Mobile application
- Web application 

### Technology stack employed
Building TerraSnap the following tech stack was used
- ReactJs for the web version, 
- React Native for the native mobile applications for android and ios
- Spring Boot for backend
- Spring Security for protected and enhanced backend APIs security
- Relational databases
